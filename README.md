## ğŸ›  Setup & Installation

### 1. Backend Setup (`task2_backend`)
1.  **Environment Variables**: Create a `.env` file in `task2_backend/` or the project root with your Gemini API key:
    ```env
    GEMINI_API_KEY=your_actual_api_key_here
    ```
2.  **Installation**:
    ```bash
    cd task2_backend
    python -m venv venv
    .\venv\Scripts\activate
    pip install -r ../requirements.txt
    ```
3.  **Run Server**:
    ```bash
    uvicorn app.main:app --host 0.0.0.0 --port 8001 --reload
    ```

### 2. User Dashboard Setup
1.  **Installation**:
    ```bash
    cd user-dashboard
    npm install
    ```
2.  **Run Development Server**:
    ```bash
    npm run dev -- --port 3000
    ```

### 3. Admin Dashboard Setup
1.  **Installation**:
    ```bash
    cd admin-dashboard
    npm install
    ```
2.  **Run Development Server**:
    ```bash
    npm run dev -- --port 3002
    ```
